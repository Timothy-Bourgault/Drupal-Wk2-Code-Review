# _Cipher_

#### _Site that uses a custom module which encrypts and entered phrase_

#### By _**Tim Bourgault**_

## Description

* _ Input:
    Phrase:
    "Hi, my name is Tim."
    Shift Value:
    2
    Shift Direction:
    Right

    Output:
    "j k o a p c o g k u v k o"_


## Setup/Installation Requirements

* _Clone the program from it's repository at GitHub.com by searching for the user name Timothy-Bourgault and then looking for Drupal Wk2 Code Review repository_
* _Open MAMP and go into the preferences menu.. Under the Weg Server tab, change the document root folder to the upper level folder of the cloned repository_
* _Start Apache and MySQL servers_
* _Click Open WebStart page to access phpMyAdmin under the tools dropdown_
* _Once inside phpMyAdmin, select the import tab and then click the Choose File to browse your computer for the database backup inside the cloned repository_
* _Once the database backup has been imported you can then access the site by directing your browser to localhost:8888_
